GetMappedString
Get the mapped string for text labels available across the app. This method will check if their is any available mapping to Config Api data and picks the value from their. Else it will use the local resource file to set the text label. This function helps in re-using the string mapping across the app. This mapper is required because some strings are picked from api data.
